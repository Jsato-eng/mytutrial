# mytutrial

git branch -r
: リモートブランチの一覧を表示する

git branch
: ローカルのブランチを表示する

git checkout feature/#1_add_xxxx
: ローカルのブランチを切り替える

git fetch
: リモートにつくられたブランチ情報をローカルに同期
リモートの「master」ブランチ → ローカルの「origin/master」ブランチ

git merge 
: ローカルの「origin/master」ブランチ → ローカルの「master」ブランチ

git pull
: git fetch、git mergeを同時に行うコマンド
